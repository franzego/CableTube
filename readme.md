# CableTube

It is a retro-styled search engine that wraps Google's index. Trying to access really old youtube videos is quite difficult if not impossible - I am talking about 10 -15 years ago content. 

They can be found using google indexes though. This is just a wrapper over it. Decided to go for the old youtube style (circa 2008-2009) using plain html, css and JS. Why ??? *FREEWILL*.

Anyway, it is nothing too serious. Try it and have fun digging through old Youtube content.

Shoutout to serperAPI. Used them for the google search engine. Google's custom search engine API - to my dismay - is being discontinued.
